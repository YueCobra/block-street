## 🛠️ 安装

### 前提条件

- Python 3.11 或更高版本
- 其他依赖...

### 安装步骤

**方法：使用 pip（推荐）**

```bash
# 1.克隆项目
git clone https://github.com/YueCobra/block-street.git
cd block-street

# 2. 创建虚拟环境
python -m venv venv

# 3. 激活虚拟环境
# Windows:
venv\Scripts\activate
# Ubuntu:
source venv/bin/activate

# 4.普通运行
python main.py
# 配置具体邀请码 具体次数
python block_street_test.py
```
